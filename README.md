# AI Medical Report Helper

## 📌 Project Overview
This project is a web-based application built using React and TypeScript that helps analyze medical reports using AI.

## 🚀 Features
- OCR result processing
- Image input handling
- AI-based analysis
- User-friendly interface

## 🛠 Tech Stack
- React.js (TypeScript - TSX)
- AI integration
- Web-based UI

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start project:
   npm start

## 📊 Purpose
This project helps automate medical report understanding and improves efficiency.
